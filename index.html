Inputs and labels

All input elements must have a label element associated with them, primarily for accessibility reasons but also as a way to denote what content should be entered into the field.

Each input needs an id attribute, so that the label points to the id using its for attribute

Input's type attributes

There are many type of input

  • number
  • email
  • url
  • password: marks the inputted text as bullets
  • tel
  • time: accepting time - hrs, mins, secs
  • date: a calender picker
  • range: selecting from a range of numbers
  • color
  • file
  • checkbox: allow a selection of multiple items out of a few possibilities
  • radio: allow users to select a single option out of many, similar to a select box
  • search
  • min: specifying a minimum value, for ranges and numbers
  • max: specifying a maximum value, for ranges and numbers
  • step: specifying a jump value for each increment, for ranges and numbers

Button

All forms need to have buttons otherwise the form’s information can’t be sent.

The button should be set to “submit”. But if you don’t want it to submit the form and control the button with JavaScript, leave the type attribute off



Input's extra settings

Required attribute

required is added to force user to fill out an important field, like email, address, etc.

Lengths

minlength & maxlength are used to control the length of inputted text

Patterns

pattern is used to restrict the format of the entered txt

Auto-capitalization

Disable or Enable auto-capitalization in the field.

autocapitalize = "none"
autocapitalize = "off"

No autocapitalization is applied (all letters default to lowercase)


autocapitalize = "on"
autocapitalize = "sentences"

The first letter of each sentence defaults to a capital letter; all other letters default to lowercase


autocapitalize = "words"

The first letter of each word defaults to a capital letter; all other letters default to lowercase


autocapitalize = "characters"

All letters should default to uppercase

Spellcheck

spellcheck

Trigger the browser to perform spell checking in the field.

Autofocus

autofocus

When the page loads, this field will be focused. Be really careful, only use this when the whole purpose of the page is to fill the form.
